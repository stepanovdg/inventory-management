Frontend · Vue 3

Single-Page Application

  • User changes a filter in FilterBar
  • useFilters exposes shared reactive state
  • View calls api.js; axios builds ?warehouse=&category=&status=&month=
  • Raw data kept in ref; views derive UI via computed
Backend · FastAPI

REST API (:8001)

  • Route handler receives optional query params
  • apply_filters() / filter_by_month() filter in memory
  • Dashboard & reports endpoints aggregate values
  • response_model validates & serializes via Pydantic
Data · JSON

Mock Data Store

  • mock_data.py loads files once at startup
  • Held as module-level lists / dicts (no database)
  • inventory, orders, demand_forecasts, backlog_items, spending, transactions, purchase_orders
  • Read-only at runtime; restart reloads from disk
